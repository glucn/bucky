# Product Overview

**Bucky** is a personal bookkeeping software built as an Electron desktop application. It provides comprehensive financial management capabilities including:

- **Account Management**: Support for assets, liabilities, and category accounts with multi-currency support
- **Transaction Tracking**: Double-entry bookkeeping with journal entries and lines
- **Credit Card Management**: Specialized features for credit card accounts including limits, interest rates, and payment tracking
- **Investment Portfolio Management**: Track investment positions, price history, and performance
- **Account Grouping**: Organize accounts into logical groups for better management
- **Checkpoints**: Create balance reconciliation points for account verification
- **Import/Export**: CSV transaction import with automatic category creation
- **Multi-Currency**: Support for different currencies with exchange rate tracking

The application follows double-entry accounting principles where every transaction affects at least two accounts, ensuring balanced books. It's designed for personal use with a focus on simplicity and accuracy.

## Key Features

- SQLite database with automatic test/development environment separation
- Real-time balance calculations and net worth tracking
- Automated transaction categorization with fallback to default categories
- Investment tracking with cost basis methods (FIFO, Average Cost)
- Credit card metrics including utilization and minimum payments
- Account archiving and restoration capabilities
